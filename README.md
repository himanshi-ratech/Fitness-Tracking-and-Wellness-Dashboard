# Fitness-Tracking-and-Wellness-Dashboard
A Django-based fitness and wellness portal that allows users to manually track health activities and access diet plans, yoga and workout guides, nutrition information, motivational content, and progress insights through a structured dashboard. Along with an integrated chatbot for user support and guidance.
Fitspiration is a Django-based fitness and wellness web application designed to help users maintain a healthy lifestyle through guided content, manual health tracking, and motivation-focused resources. The project emphasizes a holistic approach to fitness by combining workouts, yoga, nutrition guidance, and user engagement features in a single platform.
(Note: This project is currently under development and not yet deployed.)

Key Features
1. Home Page
Introduces the platform with easy navigation to key sections such as Workout, Yoga, Diet Plans, Weight Loss, and Fitness Strategies.
2. About Page
Explains the mission, objectives, and holistic wellness approach of the platform.
3. Classes Page
Displays weekly workout and yoga class schedules to help users plan their fitness routines.
4. Blog Page
Provides fitness, nutrition, wellness tips, and motivational articles to encourage consistency and healthy habits.
5. Fitness Dashboard
Allows users to manually track health-related activities such as BMI, sleep hours, water intake, workouts, calories, and steps, along with simple progress visualization.
6. Chatbot Support
An integrated chatbot that assists users with fitness guidance, navigation help, and general wellness-related queries.
7. Authentication System
-Secure Login and Signup functionality using Django authentication
-Session-based user login for personalized access
-Error handling for invalid credentials
-“Forgot Password” and new user registration support

Technology Stack
Frontend: HTML, CSS, JavaScript
Backend: Python, Django
Database: SQLite
Data Visualization: Matplotlib
Design Focus: Clean UI/UX for easy navigation and accessibility

Objective
The objective of Fitspiration is to provide a simple, content-driven fitness platform where users can track their activities manually, access structured workout and nutrition guidance, stay motivated, and build healthier lifestyle habits.
